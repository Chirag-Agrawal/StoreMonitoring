Data Descriptions:

  1. The Store Status CSV file should have 3 columns (`store_id, timestamp_utc, status`) where status is active or inactive. All timestamps are in **UTC**.
  2. The Store hours csv should contain data `store_id, dayOfWeek(0=Monday, 6=Sunday), start_time_local, end_time_local`.
  3. The Store Timezone csv should contain `store_id, timezone_str`.
